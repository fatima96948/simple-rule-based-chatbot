
# Simple Rule-Based Chatbot

## Project Overview

This project is a simple rule-based chatbot built using Python. It uses pattern matching to respond to user inputs and includes a small knowledge base to answer basic questions.

## Features

* Responds to greetings (Hi, Hello, etc.)
* Handles small talk
* Answers basic domain questions (AI, Python, Machine Learning)
* Console-based interaction
* Logs conversation history to a file

## Technologies Used

* Python
* Built-in libraries: `random`, `re`

## How to Run

1. Install Python (if not installed)
2. Download or clone this repository:

   ```bash
   git clone https://github.com/your-username/simple-rule-based-chatbot.git
   ```
3. Run the chatbot:

   ```bash
   python chatbot.py
   ```

## Example Interaction

```
Bot: Hello! I'm a simple chatbot. Type 'bye' to exit.
You: Hi
Bot: Hello!
You: What is AI?
Bot: AI stands for Artificial Intelligence.
You: Bye
Bot: Goodbye!
```

## Output

All conversations are saved in:

* `chat_log.txt`
